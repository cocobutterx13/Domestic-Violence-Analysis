# Domestic Violence Analysis

> ⚠️ **Work in Progress** — this project is actively being developed.

## Overview
An R-based analysis of domestic violence against women, examining how 
socioeconomic and demographic factors such as education, employment, income, 
and marital status relate to vulnerability.

## Methods
- Exploratory Data Analysis (EDA)
- Statistical Modelling & Inference
- Data Visualisation

## Repository Structure
- `data/` — raw and processed datasets
- `scripts/` — R scripts for analysis
- `outputs/` — plots and results

## Notes
Part of an ongoing data analytics portfolio.
